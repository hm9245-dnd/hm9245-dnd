Steam Anti DNS Poisoning For Mac Document

因为我没有用过苹果系统，部分内容是从必应查的，如果有错误请指正

  • 安装brew（需要的库较多，耐心等待）：
  • 在终端先运行/bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)"
  • 随便选一个下载源
  • 提示安装Git时点是
  • 安装好之后再执行刚刚的指令
  • brew配置源的时候也是随便选
  • 安装完之后重启终端或者运行source /Users/（你的用户名）/.zprofile
  • 安装完brew之后开始安装NGINX：
  • brew install nginx
 
  • 安装完之后，安装wget（这个需要的库也多，也要等一下）
  • brew install wget
  • 安装完wget之后，wget配置文件、证书和hosts
  • wget wget.haoming.gq/nginx.conf
  • wget wget.haoming.gq/pixiv.conf
  • wget wget.haoming.gq/ca.zip
  • wget wget.haoming.gq/hosts
  • 解压压缩包
  • unzip ca.zip -d /usr/local/etc/nginx/
  • 复制配置文件到nginx配置文件目录
  • cp nginx.conf /usr/local/etc/nginx/ 
    cp pixiv.conf /usr/local/etc/nginx/
  • 复制hosts文件到hosts文件存放目录
  • sudo cp hosts /etc
  • 下载根证书
  • 安装证书
  • 之后在钥匙串设置证书
  • 选择你安装证书时添加的位置，找到Pixiv.net
  • 展开“信任”，“使用此证书时”选择“使用系统默认”
  • 复制完和安装完证书后启动nginx（或者想用的时候）
  • nginx
  • 如果不想用了可以使用下列命令关闭
  • nginx -s stop PS：我可能会在后续写个脚本来控制打开和关闭
