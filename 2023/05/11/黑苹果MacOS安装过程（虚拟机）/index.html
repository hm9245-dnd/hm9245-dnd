黑苹果MacOS安装过程（虚拟机）

本文章借鉴文章：

https://zhuanlan.zhihu.com/p/383415378

最详细AMD Ryzen CPU，VMware 15安装macOS 10.15.x Catalina 记录（第一篇）_city_happiness的博客-CSDN博客

  • 准备工具：
  • Mac系统镜像（本文章以10.15为例）
  • VMware虚拟机（任何版本都可）
  • unlocker
 
  • 新建虚拟机之前先用unlocker解锁
 
  • 首先，创建新虚拟机，选择自定义（高级）
 
  • VMware版本选择低于15.1的版本，不要高于，也不要太低
  • 系统选择macOS 10.15
  • 点“是”
  • 选择第三个
  • 之后按照自己硬件配置选择即可（除了硬盘要选80G之外）
 
  • 在虚拟机设置中，USB兼容性选2.0，把第一个勾勾上
  • 到选项，如下图所设置
  • （新增）将网络改回NAT，不然会没网（此步骤不确定，如果出现没网的情况则执行此步骤）
 
  • 之后（也是最重要的一环），更改虚拟机文件夹中后缀为vmx的文件
  • 使用记事本打开，将下列文本加入最后一行
  • smc.version = "0"
cpuid.0.eax = "0000:0000:0000:0000:0000:0000:0000:1011"
cpuid.0.ebx = "0111:0101:0110:1110:0110:0101:0100:0111"
cpuid.0.ecx = "0110:1100:0110:0101:0111:0100:0110:1110"
cpuid.0.edx = "0100:1001:0110:0101:0110:1110:0110:1001"
cpuid.1.eax = "0000:0000:0000:0001:0000:0110:0111:0001"
cpuid.1.ebx = "0000:0010:0000:0001:0000:1000:0000:0000"
cpuid.1.ecx = "1000:0010:1001:1000:0010:0010:0000:0011"
cpuid.1.edx = "0000:0111:1000:1011:1111:1011:1111:1111"
smbios.reflectHost = "TRUE"
hw.model = "MacBookPro14,3"
board-id = "Mac-551B86E5744E2388"
keyboard.vusb.enable = "TRUE"
mouse.vusb.enable = "TRUE" 加入完之后保存
     
  • 挂载好镜像之后就可以开启虚拟机了
  • 进入虚拟机后，选择第四项
  • 进入后，选择内置硬盘名字带有“VMware”的磁盘，右键，然后点击抹掉
  • 点击之后，格式选择AFPS，名字随便填
  • 抹掉之后，就退出
  • 回到刚开始的界面，选择第二项
  • Mac的安装过程除了格式化都和Windows差不多，这里就不做例子了，安装完之后关闭虚拟机，然后把系统改回Mac
  • 又是漫长的等待.......
  • 等待完之后，国家选中国大陆，之后的设置就按照自己的喜好选择即可
